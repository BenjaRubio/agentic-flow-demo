# Dataset Package

Esta carpeta contiene el material base para la prueba técnica.

## Estructura
- `docs/`: políticas, notas y FAQ internas
- `tickets/`: ejemplos de tickets para entrenamiento y prueba
- `tools/`: especificación de tools simuladas
- `examples/`: consultas sugeridas para demo manual

## Intención del dataset
El dataset fue diseñado para evaluar:
- manejo de conflicto entre políticas
- detección de vigencia
- criterio de escalamiento
- razonamiento sustentado en evidencia
- clasificación y priorización
- uso controlado de tools

## Nota
No todos los documentos tienen la misma confiabilidad ni vigencia.
