# SLA Matrix
Version: 1.0
Status: Active

Priority levels:
- High: response within 30 minutes
- Medium: response within 4 hours
- Low: response within 1 business day

Rules:
- Customs retention => High
- Reefer / temperature-sensitive exception => High
- Missing invoice copy => Low
- Basic tracking request => Low
- Pricing discrepancy with active shipment => Medium
