# Shipping Policy v2
Version: 2.0
Effective date: 2025-07-01
Status: Active

Standard operational requests may be answered automatically only if the assistant confidence is above 0.80 and supporting evidence is available.

Any urgent request involving delay, customs retention, or temperature-sensitive cargo must be escalated for human review.

Pricing disputes must be routed first to pricing operations, not commercial.

If no exact policy is found, the assistant must abstain and request clarification or escalate.
